Requirements to run the program.

Program is written in Python 3 and requires the following packages to be installed:
numpy

## Running the program
Running script is done by executing the following command in the terminal:
```
python3 IDS.py events.txt stats.txt 100000
```
where events.txt is the input file, stats.txt is the output file and 100000 is the number of days to simulate.

When the program asks for a second input, key in the following
```
stats2.txt
100000
```
